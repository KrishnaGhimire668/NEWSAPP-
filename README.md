## NEWSAPP - MERN Stack News Platform
This is a full-stack MERN news application.  
Users can register, log in, create news posts with images or videos, and edit or delete their own posts.  
All news posts are visible publicly.

## Tech Stack
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express.js
- Database: MongoDB (Mongoose)
- Authentication: JWT with cookies
- File upload: Multer

## Features
- User registration and login
- Authentication using JWT
- Protected routes
- Create news with image or video upload
- Edit and delete own news
- Public news feed
- Responsive user interface

## Backend

## Frontend
